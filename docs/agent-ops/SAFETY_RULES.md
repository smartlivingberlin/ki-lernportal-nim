# Sicherheitsregeln

1. Keine API-Keys in Git.
2. Keine `.env` committen.
3. Keine Kundendaten in Agenten hochladen.
4. Keine Zahlungsdaten in Agenten hochladen.
5. Kein Merge ohne menschliche Prüfung.
6. Kein Deploy ohne separaten Freigabe-Schritt.
7. Keine bezahlten Dienste ohne Kostenfreigabe.
8. Keine Behauptung, dass alle NVIDIA-Modelle technisch validiert wurden.
9. Biologie/Medizin/Finanzen/Recht nur mit klaren Haftungshinweisen.
10. Jede Agenten-PR muss klein, testbar und erklärbar sein.
