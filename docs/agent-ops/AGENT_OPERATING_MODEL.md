# Agent Operating Model

## Rollen

## ChatGPT

Strategie, Architektur, Terminal-Begleitung, Review, Prompting, Qualitätssicherung.

## Jules

GitHub-basierter Coding-Agent für kleine Branch-/PR-Aufgaben.

Geeignet für:
- Frontend-Komponenten
- Tests
- Doku
- kleine Refactorings
- API-Skeleton
- RAG-Skeleton

Nicht geeignet für:
- Rechtsentscheidungen
- Datenschutzfinalisierung
- Business-Freigaben
- Secrets
- Produktiv-Merges

## Codex

Lokales/Cloud-Coding, Refactoring, Tests, Fehleranalyse, Codebase-Fragen.

## Manus Pro

Markt-/Wettbewerbsanalyse, UX-Kritik, Strategie, externe Recherche.

## Claude/Claude Code

zweite Architekturmeinung, lange Spezifikationen, Code Review.

## WSL-Terminal

Wahrheitsebene: Build, Test, Git-Status, Security-Scan, lokale Prüfung.
